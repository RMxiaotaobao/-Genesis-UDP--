## 一句话介绍

> **让调试像看视频一样直观，让参数修改像操作仪表盘一样直接。**

Genesis UDP 调参 · 图传调试助手面向智能车调试场景，集成变量监视、波形显示、远程调参、图像传输和数据记录能力，帮助车队更快定位问题、更稳验证控制效果。

## 核心能力

| 能力 | 解决的问题 |
|------|------------|
| **实时变量监视** | 告别串口刷屏，关键变量表格化展示 |
| **波形可视化** | 支持单曲线、叠加图、仪表盘，控制状态一眼看清 |
| **双向 UDP 调参** | 不重新烧录，也能实时修改下位机参数 |
| **MJPEG 图传** | 调试画面与数据同屏，视觉效果直接验证 |
| **CSV 数据记录** | 自动保存调试数据，赛后复盘和参数分析有依据 |

## 为什么开源内测

| 目标 | 说明 |
|------|------|
| **服务车队调试** | 调试效率直接影响备赛节奏和验证质量 |
| **收集真实场景** | 不同车型、不同算法、不同网络环境都能帮助完善工具 |
| **沉淀工具链** | 把临时脚本变成可复用、可传承的车队基础设施 |

## 内测重点

- **功能验证**：变量监视、波形显示、UDP 调参、MJPEG 图传是否稳定。
- **问题捕捉**：记录崩溃、卡顿、连接异常、显示异常等问题。
- **需求收集**：欢迎提出更适合车队训练和比赛的功能建议。

## 快速上手

```bash

cd ECJTU-Genesis-UDP-Debug-Assistant
pip install -r requirements.txt
python src/variable_monitor_v3.py
```
